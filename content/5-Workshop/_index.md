# Full-Stack Deployment on AWS - Via SmartMenu Project

#### Overview

**SmartMenu** is a smart electronic menu system integrated with Artificial Intelligence (AI), built on advanced AWS Serverless architecture.

In this workshop, you will learn how to deploy and configure a complete Full-stack application using AWS cloud services to ensure high availability, security, and cost optimization.

We will approach the deployment through the core components of the solution:
+ **Frontend:** Use Amazon S3 and CloudFront to host and distribute the ReactJS application quickly with global low latency.
+ **Backend & Database:** Deploy serverless APIs with API Gateway and AWS Lambda, connecting data to Amazon DynamoDB.
+ **AI & Security:** Integrate AI models from Amazon Bedrock and set up strict IAM security permissions.
