# FCAJ Community Day 27/6/2026

### Event Objectives

- Update knowledge and new trends on AI, AWS, Amazon Q, AI Agents, and MCP (Model Context Protocol).
- Introduce AWS technologies and services with practical enterprise applications.
- Share career development experiences in the cloud computing field, from student stage to founder.
- Elevate understanding of voice AI system architecture, AWS DevOps Agents, and Amazon Q in enterprise environments.
- Provide a practical view on operation costs, security, VPC configurations, and modern cloud deployment.

- **Nhat Tran** - CTO at CloudThinker
- **Nghị Danh Hoàng Hiếu** - Renova Cloud

### Key Highlights

- **Cloud Career Development Journey**: Hearing about the journey from being a student to becoming a founder, major milestones (obtaining the first AWS certification in 2020-2021), and analysis of cloud/AI job market trends.
- **Voice AI System Architecture**: How to build natural voice agent communication systems, how system components coordinate, and practical enterprise applications in automating customer service and operation support.
- **AWS DevOps Agent**: How AI agents assist DevOps teams to automate repetitive tasks, analyze system error logs, and optimize agent operation costs, along with a live E-shop demo.
- **Amazon Q for HR**: How Amazon Q supports human resource planning, resource allocation, rapid information lookup, and recruiting process efficiency.
- **Connecting Amazon Q to MCP & Security**: Guidance on connection, VPC configuration, access control, and building comprehensive security architectures in modern cloud AI environments.

### Key Takeaways

- **Cloud & Systems Mindset**: Understanding that technology engineers need not only coding skills but also strong cloud foundations and scalable system architecture design thinking.
- **Cost Management & Security**: The importance of optimizing AWS operation costs and establishing security controls (VPC endpoints, IAM policies) when integrating AI Agents into enterprises.
- **AI Agent & MCP Insights**: Grasping how to connect Amazon Q with external tools via the MCP protocol to build safe and private enterprise AI systems.

### Event Experience

The event brought together numerous experts in the fields of Cloud Computing and Artificial Intelligence (AI), who shared practical insights into the research, design, deployment, and operation of AI systems on the AWS platform.

One of the highlights of the event was the live demonstrations of AI Voice Agent, AWS DevOps Agent, Amazon Q, and Model Context Protocol (MCP). Through real-world use cases, I gained a deeper understanding of how these technologies can be integrated to solve business challenges. I also learned more about system architecture, cloud security, cost optimization, and operational best practices in cloud environments.

In addition to expanding my technical knowledge, I had the opportunity to interact with the speakers and members of the AWS community. These discussions provided valuable insights into the latest trends in AI and Cloud Computing, as well as the skills required to meet the evolving demands of the industry. Overall, the event broadened my knowledge, strengthened my motivation to continue learning, and helped me establish a clearer career direction in the fields of Cloud Computing and Artificial Intelligence.
