### [Blog 1 - Amazon GameLift Servers - Giải pháp quản lý Dedicated Game Server trên AWS](3.1-Blog1/)
Bài viết giới thiệu tổng quan về Amazon GameLift, cách thức hoạt động của hệ thống Dedicated Game Server, các ưu/nhược điểm chính và cách dịch vụ này hỗ trợ các studio game tự động mở rộng tài nguyên (Auto Scaling) để giảm tải quản lý hạ tầng.

### [Blog 2 - Amazon CloudWatch là gì? Giám sát Game Server trên AWS Thông qua Amazon CloudWatch](3.2-Blog2/)
Giới thiệu về Amazon CloudWatch, giải thích cách dịch vụ này thu thập Metrics, quản lý Logs và gửi Alarms cảnh báo theo thời gian thực để hỗ trợ giám sát sức khỏe của Game Server và ứng phó sự cố nhanh chóng.

### [Blog 3 - Tối ưu chi phí lưu trữ Game Assets trên Amazon S3 bằng Lifecycle Policies](3.3-Blog3/)
Hướng dẫn tối ưu hóa chi phí lưu trữ các tài nguyên game (texture, 3D model, file sao lưu) thông qua S3 Lifecycle Policies, tự động chuyển đổi giữa các Storage Class khác nhau hoặc xóa các tệp tin cũ khi hết hạn.
